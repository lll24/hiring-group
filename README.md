# Sonic-Java
Implementación de un juego de sonic y sus amigos recoguiendo basura desarrollado en Java. Este proyecto se centró en aplicar los principios de la **Programación Orientada a Objetos (POO)** para modelar la lógica del juego y en la creación de una **Interfaz Gráfica de Usuario (GUI)**.

Una característica clave fue la exploración de la **programación en red** mediante Sockets, con el objetivo de convertirlo en un juego multijugador. que fue logrado con exito
